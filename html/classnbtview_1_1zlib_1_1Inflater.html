std::vector< unsigned char > &  output 
)
inline

do_inflate() inflates the compressed data input and copies uncompressed data to the output.

Returns Z_DATA_ERROR: if the input data is corrupt.

Returns Z_OK: if it has inflated all of the data input without reaching the end of the compressed data stream.

Returns Z_STREAM_END: if it has reached the end of the compressed data stream.

Returns Z_NEED_DICT, Z_STREAM_ERROR, Z_MEM_ERROR, Z_BUF_ERROR: for various other conditions under which inflation cannot continue.

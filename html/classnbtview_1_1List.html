Detailed Description

List is a sequence container that holds Tags of a particular type.

The type of values being contained is determined at constuction and is available from List::list_type()

List exposes access to its Tag contents via methods analogous to std::vector.

In addition, if you already know the list's value type, you may want to access elements via List::get<T>(pos).
